# GeoJSON Visualizer

A premium, interactive GeoJSON editor and visualizer built with React, Leaflet, and Monaco Editor.

## Features

- **Real-time Visualization**: Map updates instantly as you type GeoJSON.
- **Pro Editor**: Powered by Monaco (VS Code engine) with syntax highlighting and auto-formatting.
- **Smart Bounding**: Map automatically zooms and pans to fit your GeoJSON data.
- **Error Resilience**: Validates JSON on the fly and shows helpful error messages without crashing.
- **Responsive Design**: Modern dark theme with a collapsible sidebar for maximum map space.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Technologies

- React 18
- Leaflet.js (Map rendering)
- Monaco Editor (JSON editing)
- Lucide React (Icons)
- Vite (Build tool)
